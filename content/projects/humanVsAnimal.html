En este proyecto personal, aplico técnicas avanzadas de aprendizaje automático junto con Azure Functions y Python para predecir los resultados de enfrentamientos ficticios a puño limpio contra 15 animales distintos.

He desarrollado tres modelos especializados de machine learning, uno para cada posible resultado—ganar, perder o resultado incierto—, permitiéndome evaluar con precisión las probabilidades en cada escenario.

Dada la complejidad de analizar múltiples especies y resultados, el proceso de predicción puede requerir tiempo adicional, lo que destaca la eficacia del sistema en manejar tareas de clasificación múltiple de manera efectiva.
